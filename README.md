# PAG S

Sitio independiente sobre cultura de paz, convivencia y prevención de violencia.

## Incluye

- `index.html`
- `styles.css`
- `script.js`
- `assets/`

## Qué hace diferente a esta versión

- Usa ilustraciones locales descargables.
- Tiene secciones dinámicas con consejos que cambian al hacer clic.
- Presenta una navegación más clara y un diseño más visual.

## Vista local

Abre `index.html` directamente o sirve la carpeta con cualquier servidor estático.

## Nota

Este proyecto está separado del resto del workspace para no mezclar identidades ni archivos.
